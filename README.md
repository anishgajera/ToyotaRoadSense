**gatsby-setup**

**Setup:**

```
git clone https://github.com/paljs/gatsby-admin-template.git

cd gatsby-admin-template

yarn install

yarn dev
```

**Docker:**

```
git clone https://github.com/paljs/gatsby-admin-template.git

cd gatsby-admin-template

docker build -t gatsby-admin-template

docker run --rm -d -p 80:80 gatsby-admin-template
```
